# Welcome to ReBlock
ReBlock is the best tool for bypassing system restrictions on your Windows or macOS device.

# What can ReBlock do?
- Bypass User Account Control messages when opening apps that need administrator (Windows) ([appUnblocker](https://github.com/yourworstnightmare1/appUnblocker))
- Force open apps from unidentified developers (macOS) ([appUnblocker](https://github.com/yourworstnightmare1/appUnblocker))
- Force show hidden files (Windows/macOS)
- Spoof bundle IDs to bypass app blocking software (macOS) ([packageSpoofer](https://github.com/yourworstnightmare1/packagespoofer))
- Install apps from package installers without administrator (macOS) ([packageExpander](https://github.com/yourworstnightmare1/packageexpander))

# Compatibility
## Global
You must have **PowerShell 5.1 or later** to for ReBlock to run properly.

## Windows
You must have **Windows 10 or newer** on an **x86** or **x64** processor for ReBlock to run properly.

## macOS
You must have **macOS 12 (Monterey) or newer** on an **x86**, **x64**, or **ARM64** processor for ReBlock to run properly.
